## 1. 브랜드 무드

| 키워드                    | 설명                                                          |

| **따뜻함 (Warmth)**       | 반려동물과 보호자의 감정적 유대를 색과 어조로 표현            |

## 2. 컬러 팔레트

### Primary — Amber Orange (브랜드 대표색)

```
#F59E0B  →  #FB923C  →  #F97316
```

## 4. 주요 컴포넌트 패턴

### 4-1. 히어로 카드 (Hero Card)

- 둥근 모서리: `rounded-3xl`
- AAmber-Orange 그라디언트 배경 + 흰색 반투명 장식 원
- 내부에 잔액, 서브 지갑 칩, 액션 버튼 통합 배치
- 텍스트 색: 전부 흰색 계열 (`text-white`, `text-white/70`)

### 4-2. 카드 컨테이너 (White Card)

- `bg-white rounded-2xl shadow-sm`
- 내부 패딩: `p-4` 또는 `px-5 py-4`
- 카드 간 간격: `gap-3`

### 4-3. 액션 버튼 (Action Button)

- CTA: `rounded-xl text-white font-semibold` + amber 그라디언트
- 히어로 내부 버튼: 흰색 반투명 글래스 (`rgba(255,255,255,0.22)`)
- 보조 버튼: `border-2 border-amber-200 text-amber-700 bg-[#FFFBEB]`
- 파괴적 액션(출금): red-orange 그라디언트

### 4-4. 뱃지 & 칩 (Badge / Chip)

- Pill 형태: `rounded-full text-xs font-semibold px-2.5 py-0.5`
- 선택 상태: amber border + amber 그라디언트 배경
- 비선택 상태: `border-gray-200 bg-white text-gray-500`

### 4-5. 바텀 시트 모달 (Bottom Sheet)

- `fixed inset-0 bg-black/50` 딤 레이어
- `rounded-t-3xl bg-white` 슬라이드업 시트
- 헤더: 아이콘 + 제목 + X 닫기 버튼
- 컨펌 버튼: 액션 유형별 그라디언트

### 4-6. 프로그레스 바

- `h-2 bg-gray-100 rounded-full overflow-hidden`
- 내부 바: 액션 유형별 컬러 + `transition-all duration-700`

### 4-7. 빈 상태 (Empty State)

- 이모지 + 2줄 한국어 안내 텍스트
- 중앙 정렬, 회색 계열

## 5. 레이아웃 & 그리드 시스템

### 전체 구조

```
데스크탑: 좌측 사이드바(w-64) + 우측 콘텐츠 영역
모바일:   상단 헤더 + 스크롤 콘텐츠 + 하단 네비게이션 바(fixed)
```

### 브레이크포인트 전략

| 구간                  | 클래스           | 레이아웃             |
| --------------------- | ---------------- | -------------------- |
| `< sm (640px)`        | 기본             | 1열 수직 스크롤      |
| `sm ~ md (640~768px)` | `sm:grid-cols-2` | 2열 그리드           |
| `md+ (768px~)`        | `md:` prefix     | 사이드바 + 고정 높이 |

### 콘텐츠 최대 너비

- 계좌/저축 단일 컬럼 페이지: `max-w-3xl mx-auto`
- 대시보드 그리드: `max-w-4xl mx-auto`

## 6. 내비게이션 구조

### 하단 탭 (모바일 5개)

```
홈 — 가계부 — 페이 — 리포트 — 마이
```

- 활성 탭: amber 아이콘 + amber-50 pill 배경
- 비활성 탭: gray-400

### 사이드바 (데스크탑 7개)

```
홈 / 가계부 / 페이 / 저축 / 리포트 / 소모품 / 마이
```

- 활성 항목: amber-orange 그라디언트 + 흰색 텍스트 + 우측 화살표

### 헤더

- 좌: 모바일 로고 (pawprint 아이콘) + 페이지 제목
- 우: 페이지별 액션 버튼 + 벨 알림 + 아바타 (`R` 이니셜)

## 7. 페이지별 UX 특징

### 홈 (Dashboard)

- 반려동물 필터 탭 (전체 / 초코🐶 / 나비🐱)
- 선택된 반려동물에 따라 **모든 카드 데이터가 실시간 필터링**
- 내 지갑 히어로 카드에서 충전·송금·결제로 바로 이동
- 2열 벤토 그리드: 내 지갑 / 지갑 현황 / 저축 목표 / 예정 지출 + 전폭 최근 거래

### 페이 관리 (AccountsPage)

- 고정 높이, 페이지 자체 스크롤 없음
- 결제 내역만 내부 스크롤
- 히어로 카드: 지갑 선택 칩 + 잔액 + 충전·출금·결제 버튼 통합
- 결제 버튼 → QR 코드 바텀 시트 (5분 만료 안내)
- 모바일 전용 FAB: 하단 중앙 amber 원형 QR 버튼

### 가계부 (Ledger)

- 월간 달력 인터랙션 — 날짜 클릭 → 사이드바에 해당일 내역
- 날짜별 지출 합계 캘린더 내 미리보기
- 예산 현황 / 월말 예상 지출 요약 카드

### 저축 (Savings)

- 수위 게이지(Water Level) 컴포넌트로 진행률 시각화
- 보라색 테마 전용 페이지
- AI 저축 추천 배너 (NEW 뱃지)
- 목표 펼치기/접기 인터랙션

### 리포트 (Reports)

- 2×2 벤토 그리드 (데스크탑) / 세로 스크롤 (모바일)
- 총 지출 도넛 차트 + 카테고리 범례
- 구성원 기여도 랭킹 (트로피/메달 아이콘)
- 다음달 AI 예상 지출 + 품종 평균 대비 비교

## 8. 아이콘 & 이모지 사용 원칙

- 아이콘 라이브러리: **Lucide React** (stroke 기반, 일관된 선 굵기 `strokeWidth={1.5}~{2}`)
- 기능 아이콘 크기: `size={14~16}` (버튼 내부), `size={18~20}` (탭/헤더)
- **이모지**: 반려동물(🐶🐱🐾), 카테고리(🏥🦴✂️💊🛒), 금융(💳💰🐖) 전반 사용, 단 기존의 아이콘이 있을 경우 교체하지는 말 것 당부
  - 카드 및 리스트 아이템에 감성적 포인트로 활용

## 10. 보이스 & 톤 (텍스트 어조)

- **간결하고 친근한 한국어**: 존댓말 기반, 불필요한 설명 생략
- **숫자는 한국식 단위**: `원`, `만` 축약 (`+1만`, `150,000원`)
- **안내 문구**: `~할 수 있어요`, `~해보세요` 부드러운 권유형
